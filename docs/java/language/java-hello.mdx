# Java Hello World Reference

## Summary

Minimal reference to compile and run a Java Hello World program without Maven or Gradle.

## Syntax / CLI

```bash
javac HelloWorld.java
java HelloWorld
```

## Parameters

| Parameter | Type | Required | Description |
| --- | --- | --- | --- |
| `HelloWorld.java` | file | Yes | Java source file to compile |
| `HelloWorld` | class | Yes | Compiled class name used by the JVM |

## Example code

File: `examples/java-hello/HelloWorld.java`

```java
public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Hello, World!");
  }
}
```

## Examples

### Basic run

Requirements: **JDK 17+ (or 11+)** installed and available in your `PATH`.

In a terminal, inside the cloned project:

```bash
cd examples/java-hello
javac HelloWorld.java
java HelloWorld
```

Expected output:

```text
Hello, World!
```

## Compatibility notes

- Works with JDK 11+ (tested with modern JDKs).
- No build tool required.

## Common errors

- `javac: command not found`: Java is not installed or not in `PATH`.
- `Could not find or load main class HelloWorld`: compile and run commands are being executed from the wrong directory.
